# Excel-Bike-Sales-Dashboard-Project
This project focuses on building an interactive Excel dashboard using European bike sales data. The goal was to clean raw data, analyze customer purchasing behavior, and present insights through clear and meaningful visualizations.
This project demonstrates practical skills in data cleaning, analysis, and dashboard creation using Microsoft Excel, making it suitable for entry-level data analyst portfolios.

---
## Objectives
1.Clean and prepare raw sales data for analysis
2.Analyze bike purchasing patterns across different customer segments
3.Build an interactive dashboard to visualize key insights

---
---
## Data Cleaning & Preparation

The following data cleaning steps were performed:

1.Removed duplicate records
2.Eliminated empty and invalid rows
3.Renamed columns for better clarity and consistency
4.Standardized data formats where required

---
---
## Data Analysis

Using Excel analysis features, the following insights were generated:

1.Bike purchases by country
2.Bike purchases by different age groups
3.Bike purchases by income groups
4.Average income of customers who purchased bikes
5.Comparison of purchasing behavior across demographics
